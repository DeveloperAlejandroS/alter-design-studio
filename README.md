# ALTER Studio — Ecosistema Digital 2026

> *"No diseñamos para encajar. Diseñamos para transformar."*

Repositorio oficial de la plataforma digital de **ALTER** — un estudio boutique basado en Bogotá que opera en la intersección de la narrativa estratégica y el rigor estético.

## Filosofía de marca

ALTER entiende el diseño como una herramienta de construcción y poder. La plataforma se articula bajo tres pilares:

| Pilar | Definición |
| :--- | :--- |
| **Atemporalidad** | Estética editorial que trasciende tendencias efímeras. |
| **Intención** | Cada píxel y cada micro-interacción responde a un objetivo estratégico. |
| **Sensibilidad** | Equilibrio milimétrico entre la calidez orgánica y la precisión arquitectónica. |

## Sistema de diseño

### Paleta cromática — Brandbook 2026

| Tono | Hex | Aplicación |
| :--- | :--- | :--- |
| Dusty Rose | `#E8D5D6` | Superficie base y atmósfera |
| Deep Purple | `#52284E` | Tipografía principal y peso visual |
| Violet Dark | `#40223D` | Acentos y estados de interacción |
| Warm Gray | `#B9ADAD` | Estructura secundaria y sutilezas |

### Tipografía

**Cormorant Garamond** (Serif) — carácter editorial, elegante y afilado.

- Display: Bold Italic — para narrativa y titulares
- Body: Light 300 — para máxima legibilidad y aire

## Entorno de desarrollo

```bash
# Clonar el ecosistema

# Instalar dependencias
npm install

# Lanzamiento en local
npm run dev

# Compilación para producción
npm run build
```
